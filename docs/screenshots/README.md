# Screenshots Placeholder

This directory contains screenshots for the README.

To add actual screenshots:

1. Run the application in development mode:
   ```bash
   npm run dev
   ```

2. Navigate to each page and take screenshots:
   - Dashboard: http://localhost:3000/dashboard
   - Products: http://localhost:3000/products
   - Orders: http://localhost:3000/orders
   - Analytics: http://localhost:3000/analytics
   - Customers: http://localhost:3000/customers

3. Save screenshots with these names:
   - `dashboard.png`
   - `products.png`
   - `orders.png`
   - `analytics.png`
   - `customers.png`

4. Place them in this directory

**Recommended screenshot settings:**
- Resolution: 1920x1080 or higher
- Format: PNG
- Browser: Chrome/Edge with DevTools closed
- Theme: Light mode for better visibility
- Zoom: 100%

**Tips for better screenshots:**
- Use sample data (run `npm run prisma:seed` in backend)
- Capture full page or key sections
- Ensure UI is in a clean state (no loading spinners)
- Include some data in tables/charts
